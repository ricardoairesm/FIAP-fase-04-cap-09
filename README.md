# Smart Wallet - FIAP Fase 04 Cap 09

Projeto de aplicação web responsiva de uma carteira digital (Smart Wallet) desenvolvido com HTML, Bootstrap 5 e Chart.js para a atividade do capítulo 09 da fase 04 da FIAP.

## Descrição do Projeto

Smart Wallet é uma interface web moderna inspirada em aplicativos de banco digital, oferecendo uma experiência de usuário intuitiva para gerenciamento financeiro pessoal. O projeto foi desenvolvido com foco em responsividade mobile-first e design minimalista.

## Tecnologias Utilizadas

- **HTML5** - Estrutura semântica da aplicação
- **CSS3** - Estilos personalizados mínimos
- **Bootstrap 5.3.2** - Framework CSS para responsividade e componentes
- **Chart.js 4.4.0** - Biblioteca para visualização de dados em gráficos
- **JavaScript (Vanilla)** - Interatividade e configuração dos gráficos

## Funcionalidades

### 1. Dashboard Principal
- **Card de Saldo**: Exibição destacada do saldo disponível
- **Ações Rápidas**: 4 botões para operações principais (Pix, Boleto, Cartão, Transferência)

### 2. Visualização de Dados
- **Gráfico de Movimentações**: Linha temporal mostrando a evolução do saldo nos últimos 7 dias
- **Últimas Transações**: Lista com as 4 transações mais recentes, incluindo ícones, descrição, data/hora e valores

### 3. Metas Financeiras
- **Acompanhamento de Objetivos**: Cards com barras de progresso visual
- Exemplos: Viagem para Europa (40%) e Fundo de Emergência (75%)
- Indicação de valor atual, meta e prazo

### 4. Investimentos
- **Resumo de Investimentos**: Card destacado com saldo total investido
- **Rendimento Mensal**: Exibição de ganhos percentuais e em reais
- **Detalhamento por Tipo**: Tesouro Direto, CDB e Renda Variável com indicadores visuais de performance

### 5. Gerenciamento de Cartões
- Visualização de cartão virtual existente
- Opção para adicionar novos cartões
